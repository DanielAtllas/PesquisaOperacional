Pesquisa Operacional - Revisão sobre Matriz

Antes de iniciarmos nosso estudo sobre pesquisa operacional, é importante retornarmos a matéria de álgebra linear e seu conteúdo sobre matrizes e suas operações. Diante disso, iremos partir do princípio e entender um pouco melhor como funciona o processo de criação de uma triz, o qual se dá no desenho/construção de uma tabela com N colunas e N linhas, as quais possuem posições e valores para cada elemento. Dessa forma, temos que uma matriz possui tamanho NxN ou NxM.

  • Matriz NxN
            3x3 = | 1  2  3 |
                  | 4  5  5 |
                  | 7  8  9 |
  • Matriz NxM
            3x2 = | 1  2 |
                  | 4  5 |
                  | 7  8 |

Soma de matrizes e multiplicação por escalar

No processo de soma entre duas ou mais matrizes, dizemos que a operação a ser realizada deverá ocorrer entre matrizes de mesmo tamanho. Dessa forma, seus elementos são calculados por posição copatível, como no exemplo a seguir:

  • Am + Bm
            A  = | 1  2 |
                 | 3  4 |
                 
                 
            B  = | 5  6 |
                 | 7  8 |
                 
                 
            A+B  = | (1+5)  (2+6) |
                   | (4+7)  (5+8) |
                   
            
            A+B  = | 6   8  |
                   | 10  12 |

Já no processo de multiplicação por escalar, a a operação é realizada de forma única para cada elemento, ou seja, é multiplicado o escalar por cada elemento de forma bem simples, da seguinte maneira:

  • Am - 2.Bm
            A  = | 1  2 |
                 | 3  4 |
                 
                 
            2.B  = | (2.5)  (2.6) |
                   | (2.7)  (2.8) |
                 
            
                   | 1  2 |  -  | 10  14 |
                   | 3  4 |     | 18  16 |
            
            
            A-2.B  = | -9   12 |
                     | 15   11 |

Diante deste conceito, temos então o seguinte teorema: Sejam A, B e C matrizes de mesmo tamanho, e sejam "r" e "s" escalares, temos:

  • A+B = B+A

  • (A+B)+C = A+(B+C)

  • A+0 = A

  • r.(A+B) = (r.A)+(r.B)

  • (r+s).A = (r.A)+(s.A)

  • r.(s.A) = (r.s).A

Multiplicação de matrizes

No caso da necessidade de multiplicação entre duas matrizes, de tamanhos diferentes temos a regra de que a quantidade de colunas de Am deve ser igual a quantidade de linhas de Bm, ou seja, a multiplicação só poderá ser aplicada neste caso, gereando assim uma matriz resultante Rm. Para isso então, coletamos os elementos da matriz Am e multiplicamos pelas colunas de Bm, do seguinte modo

            A  = | 2   3 |
                 | 1  -5 |
                 
                 
            B  = | 4   3  6 |
                 | 1  -2  3 |
                 
                 
    Logo, a quantidade de colunas de Am é exatamente a mesma quantidade de linhas de Bm. Por tanto, realizaremos a multiplicação:
            
            A.B  = | 2   3 |  *  | 4 |  =  | (2.4)+( 3.1) |  =  | 11 |
                   | 1  -5 |     | 1 |     | (1.4)+(-5.1) |     | -1 |
            
                   | 2   3 |  *  |  3 |  =  |  0 |  
                   | 1  -5 |     | -2 |     | 13 |
                   
                   | 2   3 |  *  | 6 |  =  | 21 |  
                   | 1  -5 |     | 3 |     | -9 |
                   
                   
            A.B  = | 11   0  21 |
                   | -1  13  -9 |

Diante disso, temos a implementação do seguinte teorema: Se Am é matriz a ser multiplicada por B e C, de tamanhos corretos, temos:

  • A(B.C) = (A.B).C

  • A.(B+C) = (A.B) + (A.C)

  • r.(A.B) = (r.A).B = A.(r.B)

Divisão de matrizes

Por fim, temos o processo de divisão entre matrizes, onde é calculado o produto da matriz Am pela inversa da matriz Bm, do seguinte modo:

  • Am / Bm
            A/B = A.B^-1
                 
                 
            B  = | 3  4 |
                 | 2  5 |
                 
            B^-1  = | 3  4 |  *  | a  b |  = | 1  0 | 
                    | 2  5 |     | c  d |    | 0  1 | 
                    
            
    Diante disso, temos:

            (3.a) + (4.b) = 1   (2.a) + (5.b) = 0

            A.B^-1  = | 1  2 |  *  | a  b |  =  A/B
                      | 3  4 |     | c  d |
